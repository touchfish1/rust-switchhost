# 应用图标

请在此目录下放置以下图标文件：

- `icon.ico` - Windows 应用图标 (推荐 256x256)
- `icon.icns` - macOS 应用图标
- `32x32.png` - 32x32 PNG 图标
- `128x128.png` - 128x128 PNG 图标
- `128x128@2x.png` - 256x256 PNG 图标（高分辨率）
- `icon.png` - 通用 PNG 图标（用于系统托盘）

## 图标设计建议

- 使用简洁的图标设计，代表 hosts 文件管理
- 建议使用网络/域名相关的图标元素
- 确保在小尺寸下清晰可辨

## 临时方案

在开发阶段，可以使用任意 PNG 图片作为占位符。
