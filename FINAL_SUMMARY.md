## 🚀 如何运行

### 开发模式

```bash

# 2. 安装依赖（如果还没安装）
```


## 🎊 特别说明

### 沙箱环境限制
在 Trae IDE 沙箱环境中运行时，应用会遇到权限限制：
- 无法访问 `%APPDATA%` 目录
- 无法修改 hosts 文件

**解决方案**：在本地非沙箱环境中运行即可正常使用

### 权限要求
- **Windows**: 需要管理员权限修改 hosts 文件
- **macOS/Linux**: 需要 sudo 权限
